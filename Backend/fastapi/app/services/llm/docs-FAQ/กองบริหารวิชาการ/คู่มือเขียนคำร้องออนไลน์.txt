คำร้องขอสำรองที่นั่งออนไลน์
1. เข้าสู่ระบบ https://reg.su.ac.th
2. ระบบเปิดบริการให้นักศึกษาสามารถยื่นคำร้องออนไลน์ได้ที่ เมนู "เขียนคำร้อง"
3. เงื่อนไขการขอสำรองที่นั่ง
3.1 รายวิชาที่สำรองที่นั่งเต็มแล้ว
3.2 รายวิชาที่ไม่ได้สำรองที่นั่งให้กับนักศึกษาในกลุ่มนั้น
3.3 ไม่สามารถยื่นคำร้องรายวิชาเดียวกันซ้ำได้ หากยังมีคำร้องรายวิชาเดียวกันอยู่ระหว่างพิจารณาอนุมัติ
3.4 ไม่สามารถยกเลิกคำร้องที่ผ่านการพิจารณาอนุมัติแล้วได้
***ขั้นตอนการพิจารณาอนุมัติคำร้อง นักศึกษายื่นคำร้อง -> อาจารย์ผู้สอน พิจารณาอนุมัติ -> คณะวิชา พิจารณาอนุมัติ -> กองบริหารงานวิชาการ พิจารณาอนุมัติ
4.เลือกเมนู "คำร้องขอสำรองที่นั่ง" เพื่อเข้าสู่หน้าจอยื่นคำร้อง
5. เข้าสู่หน้าจอยื่นคำร้อง
5.1 ตรวจสอบข้อมูล ระบบจะดึงข้อมูลเบอร์โทร และอีเมลจากฐานข้อมูลประวัตินักศึกษามาแสดงนักศึกษาสามารถแก้ไขเบอร์โทรศัพท์ได้ แต่จะไม่สามารถแก้ไขอีเมลได้
5.2 ระบุสาเหตุ/เหตุผล ที่เขียนคำร้องขอสำรองที่นั่ง
5.3 ตรวจสอบความถูกต้อง และคลิกปุ่ม "ถัดไป"
6. เข้าสู่หน้าจอ เลือกรายวิชาที่จะสำรองที่นั่ง
6.1 ระบุรหัสรายวิชา และค้นหาข้อมูลรายวิชาที่ต้องการจะสำรองที่นั่ง
6.2 ระบบแสดงรายวิชาตามที่ระบุในขั้นตอนที่ 6.1 จากนั้นคลิกปุ่ม "เลือก" ด้านหน้ารายวิชาที่ต้องการยื่นคำร้องขอสำรองที่นั่ง ระบบจะนำรายวิชาดังกล่าวไปแสดงในพื้นที่ "รายวิชาที่ต้องการสำรองที่นั่ง" ให้โดยอัตโนมัติ โดยระบบจะตรวจสอบเงื่อนไขการยื่นคำร้องขอสำรองที่นั่ง หากรายวิชาที่เลือกไม่ผ่านเงื่อนไขจะแสดงข้อความแจ้งเตือน ให้ทำการลบรายวิชาที่ไม่ผ่านเงื่อนไขออก โดยคลิกปุ่ม "ลบ" ด้านหน้ารายวิชา ให้เหลือเฉพาะรายวิชาที่สามารถยื่นคำร้องได้
6.3 คลิกปุ่ม "ยื่นคำร้อง" เพื่อยื่นคำร้อง และเข้าสู่หน้าจอยืนยันคำร้อง
6.3 คลิกปุ่ม "ยืนยัน" เพื่อยืนยันการยื่นคำร้อง
7. เมื่อคลิกยืนยันส่งคำร้อง เป็นอันเสร็จสิ้นกระบวนการยื่นคำร้อง สามารถคลิกปุ่ม "ตรวจสอบคำร้อง" เพื่อเข้าสู่หน้าจอตรวจสอบคำร้องได้
***หากคำร้องของนักศึกษาได้รับการ "อนุมัติ" ระบบจะแจ้งผลให้ทราบทางอีเมล หรือสามารถติดตามและตรวจสอบผลได้จากเมนู ตรวจสอบคำร้อง นักศึกษาควรรีบดำเนินการลงทะเบียนเพื่อป้องกันที่นั่งที่ขอสำรองไว้ "เต็ม"

คำร้องขอติด W ออนไลน์
1. เข้าสู่ระบบ https://reg.su.ac.th
2. ระบบเปิดบริการให้นักศึกษาสามารถยื่นคำร้องออนไลน์ได้ที่ เมนู "เขียนคำร้อง"
3. เงื่อนไขคำร้องขอติด W
3.1 ไม่สามารถถอน W ได้หากนักศึกษามีหนี้ค้างชำระ
3.2 ไม่สามารถถอน W ได้ กรณีรายวิชากลุ่มนั้นไม่มีผู้สอน
3.3 รายวิชาที่ยื่นคำร้องสถานะเป็น "รออนุมัติ" หรือยังดำเนินการไม่เสร็จสิ้นตามขั้นตอนการพิจารณาอนุมัติ นักศึกษาจะไม่สามารถยื่นคำร้องรายวิชาเดิมซ้ำได้
3.4 นักศึกษาจะยกเลิกรายการได้ ก็ต่อเมื่อยังไม่มีการอนุมัติ
***ขั้นตอนการพิจารณาอนุมัติคำร้อง นักศึกษายื่นคำร้อง -> อาจารย์ผู้สอน พิจารณาอนุมัติ -> คณะวิชา พิจารณาอนุมัติ -> กองบริหารงานวิชาการ พิจารณาอนุมัติ
4.เลือกเมนู "คำร้องขอติด W" เพื่อเข้าสู่หน้าจอยื่นคำร้อง
5. เข้าสู่หน้าจอยื่นคำร้อง
5.1 ตรวจสอบข้อมูล ระบบจะดึงข้อมูลเบอร์โทร และอีเมลจากฐานข้อมูลประวัตินักศึกษามาแสดงนักศึกษาสามารถแก้ไขเบอร์โทรศัพท์ได้ แต่จะไม่สามารถแก้ไขอีเมลได้
5.2 ระบุสาเหตุ/เหตุผล ที่เขียนคำร้องขอติด W
5.3 ตรวจสอบความถูกต้อง และคลิกปุ่ม "ถัดไป"
6. เข้าสู่หน้าจอ เลือกรายวิชาที่จะขอติด W
6.1 ระบบแสดงรายวิชาตามที่นักศึกษาได้ลงทะเบียนไว้ จากนั้นคลิกปุ่ม "เลือก" ด้านหน้ารายวิชาที่ต้องการถอน W ระบบจะนำรายวิชาดังกล่าวไปแสดงในพื้นที่ "รายวิชาที่ต้องการขอติด W" ให้โดยอัตโนมัติ โดยระบบจะตรวจสอบเงื่อนไขการยื่นคำร้องขอติด W หากรายวิชาที่เลือกไม่ผ่านเงื่อนไขจะแสดงข้อความแจ้งเตือน ให้ทำการลบรายวิชาที่ไม่ผ่านเงื่อนไขออก โดยคลิกปุ่ม "ลบ" ด้านหน้ารายวิชา ให้เหลือเฉพาะรายวิชาที่สามารถยื่นคำร้องได้
6.2 คลิกปุ่ม "ยื่นคำร้อง" เพื่อยื่นคำร้อง และเข้าสู่หน้าจอยืนยันคำร้อง
6.3 คลิกปุ่ม "ยืนยัน" เพื่อยืนยันการยื่นคำร้อง
7. เมื่อคลิกยืนยันส่งคำร้อง เป็นอันเสร็จสิ้นกระบวนการยื่นคำร้อง สามารถคลิกปุ่ม "ตรวจสอบคำร้อง" เพื่อเข้าสู่หน้าจอตรวจสอบคำร้องได้
***หากคำร้องของนักศึกษาได้รับการ "อนุมัติ" ระบบจะแจ้งผลให้ทราบทางอีเมล หรือสามารถติดตามและตรวจสอบผลได้จากเมนู ตรวจสอบคำร้อง

ตรวจสอบคำร้อง
1. เข้าสู่ระบบ https://reg.su.ac.th
2.  หลังจากที่นักศึกษายืนยันส่งคำร้องเรียบร้อยแล้ว นักศึกษาสามารถติดตามและตรวจสอบสถานะคำร้องได้จากเมนู "เขียนคำร้อง" เลือก "ตรวจสอบคำร้อง" โดยสามารถตรวจสอบได้จากแถบ "คำร้องที่รอดำเนินการ" และแถบ "คำร้องที่ดำเนินการเสร็จแล้ว" โดยสถานะการอนุมัติคำร้อง จำแนกได้ดังนี้
ยื่นคำร้อง => คำร้องที่รอการอนุมัติ
อนุมัติโดยผู้สอน => คำร้องที่ผ่านการอนุมัติ โดยอาจารย์ผู้สอน
อนุมัติโดยที่ปรึกษา => คำร้องที่ผ่านการอนุมัติ โดยอาจารย์ที่ปรึกษา
อนุมัติโดยวิชาการคณะ => คำร้องที่ผ่านการอนุมัติ โดยคณะวิชา
ไม่อนุมัติโดยผู้สอน => คำร้องไม่ผ่านการอนุมัติ จากอาจารย์ผู้สอน
ไม่อนุมัติโดยที่ปรึกษา => คำร้องไม่ผ่านการอนุมัติ จากอาจารย์ที่ปรึกษา
ไม่อนุมัติโดยวิชาการคณะ => คำร้องไม่ผ่านการอนุมัติ จากวิชาการคณะ
ยกเลิกโดยนักศึกษา => คำร้องถูกยกเลิกโดยนักศึกษา
ดำเนินการสำเร็จ => คำร้องที่เสร็จสิ้นแล้ว
2.1 สามารถคลิกที่ไอคอนในช่องสถานะการอนุมัติของแต่ละคำร้องเพื่อตรวจสอบความคิดเห็นจากผู้อนุมัติคำร้อง และสถานะการอนุมัติได้
2.2 กรณีที่นักศึกษาต้องการยกเลิกคำร้องจะสามารถยกเลิกได้เฉพาะคำร้องที่อยู่ในสถานะ"ยื่นคำร้อง" ที่ยังไม่ผ่านการพิจารณาอนุมัติได้เท่านั้น โดยคลิกปุ่ม "ยกเลิกคำร้อง" ด้านขวามือหลังคำร้องที่ต้องการยกเลิก
2.3 คลิกปุ่ม "ยืนยัน" เพื่อยืนยันการยกเลิกคำร้อง ระบบจะนำคำร้องที่ยกเลิก ไปแสดงในแถบ "คำรองที่ดำเนินการเสร็จแล้ว"
3. คลิกแถบ "คำร้องที่ดำเนินการเสร็จแล้ว" เพื่อตรวจสอบสถานะคำร้องที่ดำเนินการเสร็จสิ้นแล้วได้จากคอลัมน์ "สถานะการอนุมัติ"
